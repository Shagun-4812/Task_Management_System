# InMemory Task Management System

A Python-based task management engine designed for handling task lifecycles, user quotas, and time-based scheduling.

## Core Features
- **Task Search**: Priority-based searching with creation-order tie-breaking.
- **Quota Management**: Prevents users from exceeding their assigned task limits.
- **State Tracking**: Handles Active, Completed, and Overdue task states.



## Usage
Implement the `SimpleTaskManagementSystem` interface to build custom task logic.
